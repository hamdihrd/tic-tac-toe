# Tic Tac Toe v2.0

A modern, customizable Tic Tac Toe game with dynamic grid size and score tracking.

## Features

- 🎲 **Customizable grid** - 3x3 up to 20x20
- 🏆 **Adjustable win condition** - Choose how many cells in a row to win (3 to grid size)
- 👥 **Two players** - Custom names with ✖️ and ⭕ markers
- 📊 **Score tracking** - Real-time scores
- 🎨 **Modern UI** - Gradients, animations, responsive design
- 🔄 **Reset Game** - Reset board and scores
- ✨ **Win animation** - Blinking winning cells

## How to Play

1. **Configure the game:**
   - Choose grid size (3-20)
   - Set cells needed to win (3 to grid size)
   - Enter player names

2. **Click "Start"** to begin

3. **Take turns clicking cells:**
   - Player 1: ✖️ (red)
   - Player 2: ⭕ (blue)

4. **First to align the required number of cells wins!**

5. **Use "Reset Game"** to start a new match

## Technologies

- HTML5
- CSS3 (Flexbox, Grid, animations, gradients)
- JavaScript ES6+ (Factory functions, Closures, AbortController)
